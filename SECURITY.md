# Security Policy

## Supported Version

Only the latest published release is supported with security fixes.

## Reporting A Vulnerability

Do not open a public issue for vulnerabilities involving token access, arbitrary code execution, or unintended disclosure of locally stored history.

Report security concerns through [GitHub private vulnerability reporting](https://github.com/kingslayerrq/discord-last-login/security/advisories/new). Include the affected version, reproduction steps, impact, and any suggested remediation.

The project intentionally makes no network requests and does not access Discord account tokens. Reports that show otherwise are treated as high priority.
