# Support

## Before Opening An Issue

1. Install the latest release.
2. Reload Discord with `Ctrl+R`.
3. Confirm LastSeen is enabled in BetterDiscord.
4. Open LastSeen settings and review **Live diagnostics**.
5. Disable the experimental member-list option if the issue is visual.

## Bug Reports

Use the [bug report form](https://github.com/kingslayerrq/discord-last-login/issues/new?template=bug_report.yml). Include the LastSeen version, operating system, Discord and BetterDiscord versions, reproduction steps, and relevant diagnostics.

Remove usernames, user IDs, server names, messages, and other personal information from logs and screenshots.

## Unsupported Requests

The project cannot:

- Determine whether an offline user is actually invisible.
- Recover presence history from before installation.
- Track presence while Discord is closed or disconnected.
- Guarantee compatibility immediately after every Discord client update.
