# Contributor Covenant Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code and behavior.
- Respect privacy and remove personal Discord data from reports and screenshots.
- Do not use project spaces to harass, threaten, or expose others.

## Enforcement

Project maintainers may edit or remove contributions and interactions that violate these expectations, and may temporarily or permanently restrict participation.

Report conduct concerns privately to the repository owner through their GitHub profile.

This policy is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
